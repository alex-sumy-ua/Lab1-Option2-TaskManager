# For correct building and running the program "project2" from console:
1. Unarchive your zip-file.
2. Run a console.
3. Enter: mvn clean compile assembly:single
4. After compiling shut down the console and start run.bat file.


# For correct building and running the program "project2" from "IntelliJ Idea":

1. Open 'Maven Projects' on the right side of Idea.
2. Lifecycle --> clean
3. Lifecycle --> compile
4. Plugins --> assembly --> assembly:single.
5. Create and start "run.bat".
